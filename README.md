## Installation ##

  - Install Flutter
  - Switch to dev channel `flutter channel dev`
  - Upgrade flutter `flutter upgrade`
  - Run tests `flutter test`
  - Run app `flutter run`
  
 You can use docker image ammaratef45/flutter:latest
